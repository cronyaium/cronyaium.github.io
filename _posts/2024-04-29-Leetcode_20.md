# Valid Parentheses

## Statement

括号匹配

## Solution

用栈即可。

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char>sta;
        int n = s.size();
        for (int i = 0; i < n; i++) {
            if (sta.empty()) {
                sta.emplace(s[i]);
            } else {
                bool match = false;
                char c = sta.top();
                if (c == '(' && s[i] == ')') {
                    match = true;
                }
                if (c == '[' && s[i] == ']') {
                    match = true;
                }
                if (c == '{' && s[i] == '}') {
                    match = true;
                }
                if (match) {
                    sta.pop();
                } else {
                    sta.emplace(s[i]);
                }
            }
        }
        return sta.empty();
    }
};
```
