NJU-ICS2020 PA0

1. System

Ubuntu 64 bits 20.04.3

2. Setting APT source file

sudo is frequently used.

3. Enabling more vim features

set tabstop = 2

don't set colorscheme evening

don't set softtabstop=4

4. Learning to use basic tools

The tutorial for GNU/Linux written by jyy is recommended.

when you find it difficult to understand the Linux command, use man to help you.

you can simply read the options and name to get a overview

to learn:

  1. tmux 分屏
  2. 正则表达式 《鸟哥的Linux私房菜》
  3. How to ask a question here
  4. 文件管理tar
  5. 文本处理awk sed uniq cut tr
  6. 系统监控jobs ps top kill free demsg lsof

to be more familiar:

  1. gdb
  2. makefile
  3. redirect: xargs tee
  4. 文本处理grep sort wc
  5. 文本检索less find

5. git

git log

git reset --hard $hash

git checkout $hash recommended

6. Getting Source Code for PAs

I failed to clone nemu and abstract-machine

The problem was solved by change https to git to speed up in init.sh

It is required to set the gitconfig first and maybe the --global option is not needed.

sudo is of great great importance

if you fail to do sth, try sudo

manually add the environment variables

first let’s check the init.sh

if you have knowledge about bash scripting, it is easier to understand.

what matters is the sed line and the echo line followed

$1 means what we type, such as nemu or abstract-machine

$2 depends on $1, go to check the bottom of init.sh
