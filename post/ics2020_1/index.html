5. C语言世界

内存，可以看作一个数组，单位为字节

int main() {
  int *p = (void *) 1;
  *p = 1;
}

主函数的第一步是没有问题的，指定一个无类型指针给p

但是执行第二行时就会出现段错误

定义有类型指针时，首先它是一个指针，然后它才有了类型

有类型指针可以赋值给无类型指针，反之不行，需要强制类型转换

Slide P17

void printptr(void *p) {
  printf("p = %p;  *p = %016lx\n", p, *(long *)p);
}
int x;
int main(int argc, char *argv[]) {
  printptr(main);  // 代码
  printptr(&main);
  printptr(&x);    // 数据
  printptr(&argc); // 堆栈
  printptr(argv);
  printptr(&argv);
  printptr(argv[0]);
}

Slide P18

int main(int argc, char *argv[]) {
  int (*f)(int, char *[]) = main;
  if (argc != 0) {
    char ***a = &argv, *first = argv[0], ch = argv[0][0];
    printf("arg = \"%s\";  ch = '%c'\n", first, ch);
    assert(***a == ch);
    f(argc - 1, argv + 1);
  }
}

Explanation

argc: argument count
argv: argumment vector
从命令行读入字符数组， 并且记录数组数量

其中 char *argv[v]等价于char **argv --> (char *) *argv是一个指针，指针对应地址，故占8个Byte

*f 是一个函数指针，指向main函数对应的地址
f(argc - 1, argv + 1);等价于main(...); 
用栈的方式理解argv + 1
argv -> char *x0 -> char x00
argv + 1 -> char *x1 -> char x10
中间的char *以栈的形式存储，栈底为NULL
argv + 1 等价于 &argv[1]

char ***a -> (char **) *a 类型为char**，是一个指针，指向argv的地址
char *first -> (char *) first 类型为char*，是一个指针，把argv[0]赋值给了first

assert(***a == ch);类似于
if (***a == ch) {}
else {printf("Error\n");}
